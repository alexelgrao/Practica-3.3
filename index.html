Navegadores web

Un navegador web (en ingles, web browser) es un software, aplicacion o programa que permite el acceso a la web, interpretando la informacion de distintos tipos de archivos y sitios web para que estod puedan ser vistos.

La funcionalidad basica de un navegador web es permitir la visualizacion de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari: Safari es un navegador web de codigo cerrado desarrolado por apple Inc. Esta disponiblr para macOS, iPadOS e iOS.

Mozilla Firefox es un navegador web libre y de codigo abierto desarrollado para linux, android, iOS, macOS y Microsoft Windows coordinado por la corporacion Mozilla y la fundacion MOzilla.

Google Chrome: Google chrome es un navegador web de codigo cerrado desarrollado por google, aunque derivado ode proyectos de codigo abierto
